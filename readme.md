## 🛡️ CyberSecurity Tools
### Scripts usados em rotinas relacionadas a cybersegurança e OSINT em geral.

---
### Objetivo 📋
Desenvolver e disponibilizar **pequenas soluções para rotinas e atividades de cibersegurança e OSINT**, implementadas em Python ou Bash.  
Esses projetos são simples e não justificam um repositório separado, mas podem ser úteis como ferramentas independentes.  

--- 
### Scripts ⚒️
``` 
 Checkpwd ✔️
  - Verifica se sua senha consta no rockyou.txt (lista de senhas vazadas na web).
```

``` 
 Checkcpf ✔️
  - Gera ou verifica matematicamente a validade de CPFs ficticios.
```

``` 
 Find company data 🏢
 - Appcli para buscar dados de empresas no cadastro da receita.
```

``` 
 Gitget 🔍
  - Appcli para baixar projetos do github de forma automatizada
```

```
Honey pots creator 🍯
  - Esta ferramenta gera honeypots para proteger os arquivos reais de um servidor.
```

```
Ipinspector 🕵️
  - Script para buscar informações sobre um ip via terminal.
```

```
Find files 🔍
  - Scriptbash para buscar arquivos no sistema pelo nome ou extensão através de uma lista em txt
```

```
Linux gather 🕵️
  - Scriptbash (lista de comandos) que retorna informações sobre um sistema linux.
```

```
Password test 🔑
  - Script para testar a segurança de senhas através de brute force attack.
```

```
Pyencrypt 🐍
  - Appcli para criptografar e descriptografar arquivos e pastas.
```

```
Pyhash #️
  - Script para gerar hashs de arquivos de forma simplificada.
```
